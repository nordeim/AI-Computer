## 1. Tool Overview: TrustSkill v3.0

### WHAT is TrustSkill?
TrustSkill is an advanced security auditing tool specifically engineered for OpenClaw skills. It serves as a gatekeeper, identifying potentially harmful code patterns, sensitive data leaks, and architectural vulnerabilities before they can compromise the system.

### WHY use TrustSkill?
As AI agents consume and execute skills, the risk of "prompt-to-code" or "skill-to-system" attacks increases. TrustSkill provides:
- **Integrity Assurance**: Ensures third-party skills don't contain backdoors.
- **Credential Protection**: Prevents hardcoded secrets (API keys, tokens) from being leaked.
- **Injection Prevention**: Identifies dangerous sinks like `eval()` or `os.system()` that could be exploited via user input.

### HOW does it work? (Methodology)
TrustSkill utilizes a **Multi-Layer Analysis** engine:
1.  **Regex Analysis**: Rapid signature matching for known malicious patterns.
2.  **AST (Abstract Syntax Tree) Analysis**: Deep structural inspection of Python code to detect logic-based risks.
3.  **Secret Detection**: Hybrid entropy-based calculation and pattern matching for credentials.
4.  **Taint Analysis**: Data flow tracking from user input (sources) to dangerous functions (sinks).
5.  **Dependency Scanning**: Integration with the OSV database to catch vulnerable packages.
