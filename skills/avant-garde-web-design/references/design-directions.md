# Design Directions

## Aesthetic Direction Selection

Choose ONE direction and execute with precision. Bold maximalism and refined minimalism both work—key is intentionality, not intensity.

---

## Available Directions

### Brutally Minimal
- **Characteristics:** Extreme whitespace, single focal point
- **Use when:** Luxury, high-end, editorial
- **Key elements:** Near-empty layouts, massive typography, one hero element

### Maximalist Chaos
- **Characteristics:** Layered textures, bold typography, visual density
- **Use when:** Creative agencies, portfolios, art/entertainment
- **Key elements:** Overlapping elements, multiple typefaces, rich backgrounds

### Retro-Futuristic
- **Characteristics:** Neon, chrome, geometric patterns, sci-fi
- **Use when:** Tech products, gaming, innovative platforms
- **Key elements:** Gradients, glowing effects, sharp angles, dark backgrounds

### Organic/Natural
- **Characteristics:** Soft curves, earthy tones, fluid shapes
- **Use when:** Wellness, sustainability, lifestyle brands
- **Key elements:** Rounded corners, natural color palettes, flowing layouts

### Luxury/Refined
- **Characteristics:** Serif fonts, gold accents, subtle gradients
- **Use when:** Premium products, financial services, hospitality
- **Key elements:** Generous whitespace, restrained palette, elegant typography

### Editorial/Magazine
- **Characteristics:** Asymmetric layouts, bold headlines, grid disruption
- **Use when:** Content-heavy sites, news, blogs
- **Key elements:** Large display type, mixed alignments, visual rhythm

### Brutalist/Raw
- **Characteristics:** Exposed structure, monospace, high contrast
- **Use when:** Tech startups, portfolios, alternative brands
- **Key elements:** No decoration, raw HTML feel, thick borders, system fonts

### Art Deco/Geometric
- **Characteristics:** Symmetry, gold/black, stepped forms
- **Use when:** Luxury, fashion, premium experiences
- **Key elements:** Geometric patterns, metallic accents, strong vertical lines

### Soft/Pastel
- **Characteristics:** Gentle gradients, rounded shapes, light tones
- **Use when:** Lifestyle, children's products, wellness
- **Key elements:** Subtle shadows, soft colors, rounded everything

### Industrial/Utilitarian
- **Characteristics:** Raw materials aesthetic, functional focus, minimal decoration
- **Use when:** B2B software, tools, developer products
- **Key elements:** Monospace fonts, dense information, functional layouts

## Direction → Technical Commitments

| Direction | Must Have | Can Skip |
|-----------|-----------|----------|
| **Brutally Minimal** | Perfect typography, whitespace mastery | Animation, 3D, complex interactions |
| **Maximalist Chaos** | Layered animations, rich textures | Ultra-performance, simple layouts |
| **Retro-Futuristic** | Glow effects, gradients, dark mode | Traditional accessibility patterns |
| **Organic/Natural** | Soft shapes, natural palette | Sharp edges, high contrast |
| **Luxury/Refined** | Premium typography, subtle motion | Heavy animations, bold colors |
| **Editorial/Magazine** | Dynamic layouts, typography system | Predictable grids, simple hierarchy |
| **Brutalist/Raw** | Strong typography, high contrast | Decorative effects, gradients |
| **Art Deco/Geometric** | Geometric patterns, metallic accents | Organic shapes, soft edges |

## Decision Guide

### Ask These Questions:
1. **What should the user FEEL?** (Safe, excited, curious, confident?)
2. **What's the brand personality?** (Professor, founder, artist, engineer?)
3. **What would make this UNFORGETTABLE?** (One distinctive element)
4. **What's the competition doing?** (Differentiate, not imitate)

### Avoid:
- Choosing "safe" direction by default
- Mixing too many directions (pick ONE)
- Copying trends without understanding why
- Prioritizing aesthetics over function

## Component Styling by Direction

### Buttons

|-----------|-------|

| Luxury/Refined | Subtle gradient, gold accent, generous padding |
| Brutalist/Raw | Thick border, monospace, no shadow |
| Retro-Futuristic | Glowing effect, neon color, rounded |

| Direction | Style |
|-----------|-------|

| Maximalist Chaos | Overlapping, layered, multiple backgrounds |
| Editorial/Magazine | Asymmetric padding, varying widths |
| Organic/Natural | Rounded corners, soft shadows, earth tones |
