 Read through the architecture doc. The core mechanism:                                                                                                                                                                                     
                                                                                                                                                                                                                                            
 - Every message persists in SQLite (raw, never deleted)                                                                                                                                                                                    
 - Older messages get summarized into leaf nodes in a DAG                                                                                                                                                                                   
 - Leaf nodes condense upward into higher-level summaries (depth 1, 2, 3+)                                                                                                                                                                  
 - Each turn, the assembler builds context: [summaries...][recent raw messages]                                                                                                                                                             
 - Agents get retrieval tools (lcm_grep, lcm_describe, lcm_expand) to drill into any historical detail                                                                                                                                      
 - Three-level escalation: normal summary → aggressive summary → deterministic truncation fallback                                                                                                                                          
 - Integrity checks and repair utilities built in                                                                                                                                                                                           
                                                                                                                                                                                                                                            
 The database at ~/.openclaw/lcm.db would be the new persistence layer.
